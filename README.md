# PassOp - Password Manager

**PassOp** is a modern, secure, and user-friendly password management application built using React and Tailwind CSS. It allows you to store, manage, and organize your website credentials locally with ease.

## üöÄ Features

* **Secure Storage**: Save your website URLs, usernames, and passwords securely.
* **Password Visibility Toggle**: Easily show or hide passwords using the eye icon.
* **One-Click Copy**: Quickly copy your usernames or passwords to the clipboard.
* **Edit & Delete**: Manage your saved credentials by editing or removing them as needed.
* **Search & Filter**: Find your stored passwords quickly.
* **Responsive Design**: A clean UI built with Tailwind CSS that works on all screen sizes.
* **Animated UI**: Includes smooth animations powered by Lordicon.

## üì• Getting Started

Follow these steps to set up the project locally:

### Prerequisites

* Ensure you have [Node.js](https://nodejs.org/) installed.

### Installation

1. **Clone the repository**:
```bash
git clone https://github.com/anishkhan2310/passop.git

```


2. **Navigate to the project directory**:
```bash
cd PassOp

```


3. **Install dependencies**:
```bash
npm install

```



### Running the App

In the project directory, you can run:

* **`npm run dev`**: Runs the app in development mode. Open the URL provided in the terminal (usually [http://localhost:5173](https://www.google.com/search?q=http://localhost:5173)) to view it in your browser.
* **`npm run build`**: Builds the app for production to the `dist` folder.
* **`npm run preview`**: Previews the production build locally.
